# School System (Final Project)

## Author: Adrian Eduardo Guzman Loya 2538954

### Overview

This is the final project for Introduction to Programming 

The purpose of this final project is to mimic a school system using object oriented programing.
The system allows the students to register and drop a course, and allow the teacher to generat and to display scores of a course.

### Description

This project, which simulates a school system, has the following capabilities:

*Student can register course
*Student can drop course
*Courses can contain mutiple assignments with different weights
*Scores can be generated randomly for testing purposes
*The system can calculate weighted final Scores
*Course can display results in a table
